# AI Smart Test Selector

## Overview
AI system for optimizing firmware validation by selecting high-impact test scenarios.

## Features
- Test recommendation system
- ML-based failure prediction
- Streamlit dashboard

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

## How to run
1. Install dependencies:
   pip install -r requirements.txt

2. Run app:
   streamlit run src/ui/app.py
